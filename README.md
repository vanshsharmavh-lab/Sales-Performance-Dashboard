# 📊 Sales Performance Dashboard | Power BI

## 📌 Overview
This project is an interactive **Sales Performance Dashboard** built in **Power BI** to analyze sales data and provide insights into revenue, profit, customer trends, and regional performance. It enables users to monitor key business metrics and make data-driven decisions.

## 🚀 Features
- Sales & Profit KPIs
- Monthly Sales Trend
- Region-wise Sales Analysis
- Category & Product Performance
- Customer Insights
- Interactive Filters & Slicers

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📈 Key Insights
- Top-performing products and categories
- Regional sales comparison
- Monthly revenue and profit trends
- Customer purchasing patterns
- Overall business performance
